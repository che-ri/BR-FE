## 💻Use

-   React.js
-   데이터관리 : redux, redux-action(redux-actions), redux- toolkit
-   미들웨어 : redux-thunk
-   배포 : AWS S3
-   통신 : axios
-   라이브러리 : js-cookie, react-helmet
-   라우터 : connected-react-router, history
-   스타일 : styled-components, react-slick, slick-carousel,react-toastify
